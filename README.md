Bem-vindo ao repositório do projeto **"Use Mais Energia Solar"**! Este é um site desenvolvido como parte de um trabalho da faculdade com o objetivo de **conscientizar as pessoas sobre os benefícios da energia solar**.

Acesse o site ao vivo: **[usemaisenergiasolar.vercel.app](https://usemaisenergiasolar.vercel.app/)**

## 📋 Sobre o Projeto

O projeto "Use Mais Energia Solar" nasceu de um trabalho acadêmico com a missão de informar e engajar a comunidade sobre as vantagens da energia solar, tanto para o meio ambiente quanto para a economia doméstica e empresarial.

O site foi construído para ser uma ferramenta educativa, de fácil navegação, apresentando informações claras e acessíveis sobre como a energia solar funciona, seus benefícios e seu potencial.

## ✨ Funcionalidades

*   **Conteúdo Educativo:** Textos e seções dedicadas a explicar o que é energia solar e como ela pode ser aproveitada.
*   **Design Responsivo:** Interface adaptável para diferentes tamanhos de tela (celular, tablet, desktop).
*   **Interface Amigável:** Design moderno e limpo para uma melhor experiência do usuário.
*   **Hospedagem na Vercel:** Deploy contínuo e rápido através da plataforma Vercel.

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido com as seguintes tecnologias e ferramentas:

*   **[Next.js 15](https://nextjs.org/):** Framework React para renderização híbrida e geração de sites estáticos.
*   **[React 18](https://react.dev/):** Biblioteca JavaScript para construção de interfaces de usuário.
*   **[TypeScript](https://www.typescriptlang.org/):** Superset JavaScript que adiciona tipagem estática ao código.
*   **[Tailwind CSS 4](https://tailwindcss.com/):** Framework CSS utilitário para estilização rápida e responsiva.
*   **[ESLint](https://eslint.org/):** Ferramenta de análise de código para identificar padrões problemáticos.
*   **[Vercel](https://vercel.com/):** Plataforma de hospedagem para front-end e deploy contínuo.
